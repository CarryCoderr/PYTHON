In a nested if construct, you can have an if...elif...else construct inside another if...elif...else construct.
syntax
if expression1:
   statement(s)
   if expression2:
      statement(s)
   elif expression3:
      statement(s)
   elif expression4:
      statement(s)
   else:
      statement(s)
else:
   statement(s)
